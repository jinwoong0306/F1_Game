# 싱글플레이 변경사항

## 렌더링
- 맵 렌더링 범위를 카메라 뷰의 2배로 확장해 주행 시 도로 타일 팝인을 방지했습니다. (`core/src/main/java/com/mygame/f1/GameScreen.java`의 `MAP_RENDER_EXPANSION`, `updateMapRendererView()`)
- 카메라 뷰/줌은 그대로 두고 렌더 범위만 확장합니다. 멀티/서버 로직 영향 없음.

## 속도 제한
- 차량 또는 타이어 내구도가 0 이하로 떨어지면 최고 속도를 30%로 제한합니다. 피트스탑으로 내구도가 복구되면 제한이 자동 해제됩니다. (`GameScreen.limitSpeed()`)

## PIT STOP HUD
- PIT STOP 화면에서 타이어 선택 버튼과 안내 텍스트가 제목과 겹치지 않도록 하단으로 내리고 여백을 늘렸습니다. (`GameScreen.drawPitMinigameHud()`)
- 타이어 선택 슬롯을 패널 바로 위에 배치하고, `Press ENTER/SPACE` 아래에 `Q/W/E = SOFT/MEDIUM/HARD` 안내를 배치했습니다. 타이밍 바는 패널 아래(기존 `barY - 32f`) 위치 유지. 버튼 텍스트 색상은 아틀라스 텍스트라 코드만으로 변경 불가. (`GameScreen.drawPitMinigameHud()`)

## 타이어 컴파운드 성능
- 마모 시간: Soft ~90초, Medium ~130초, Hard ~150초에 100% 소모 (`getCompoundWearRate()`)
- 최고속도/회전 보정: Soft +12% 속도, Medium 기본, Hard 속도 기본·회전율 15% 감소 (`tireSpeedMultiplier`, `tireTurnMultiplier` → `limitSpeed()`, `updateSteering()`)


## 차량 속도/계기판
- 최고 속도 15% 상향(`maxForwardSpeed = 4.0f`), 가속 상승률을 느리게 조정(`speedRampUpRate = 0.03f`).
- HUD 속도 표기는 실제 속도를 0.60배 스케일하고 최대 268까지 표시되도록 변경(`HUD_SPEED_SCALE`, `HUD_SPEED_MAX`).
- 브레이크 입력: `SPACE` → `SHIFT`(좌/우)로 변경.
