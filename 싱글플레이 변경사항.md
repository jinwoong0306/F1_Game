# 싱글플레이 변경사항

# 도로 안 끊기게 렌더링 넓힘
- 맵 렌더링 범위를 카메라 뷰의 2배로 확장해 주행 시 도로 타일 팝인(늦게 그려져 끊겨 보임)을 방지했습니다. (`core/src/main/java/com/mygame/f1/GameScreen.java`에 `MAP_RENDER_EXPANSION` 상수와 `updateMapRendererView()` 적용)
- 카메라 뷰/줌 설정은 그대로 유지하며 렌더러만 더 넓게 그립니다. 성능 부담은 소폭 증가할 수 있습니다.
- 멀티플레이/서버 로직에는 영향 없습니다.

# 차량/타이어 내구도 하나라도 0이 되면 속도 1/3
- 차량 또는 타이어 내구도가 0 이하로 떨어지면 최고 속도를 30%로 제한합니다. 피트스탑으로 내구도가 복구되면 제한이 자동 해제됩니다. (`GameScreen.limitSpeed()`)
